### 默认使用的是yarn

```
vue create hello
```

- 之前使用`yarn global add @vue/cli`安装的吧？使用上面的命令创建一个vue工程，内部是使用yarn来安装组件的。
- 复习下yarn的命令

```
yarn config list
yarn config get registry
yarn config get cache-folder
```
